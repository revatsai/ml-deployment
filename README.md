# ML 模型部署 🚀

## 實作項目：

### 1️⃣ 模型訓練與自動化  
- **工具**：Google Colab
- **方法**：RandomForest   
  - 在 Google Colab 平台上進行模型訓練。

### 2️⃣ 建立 API 接口  
- **框架**：FastAPI  
  - 使用 FastAPI 建立 API 接入訓練好的 ML 模型。  
  - 接收用戶提交的數據並進行預測。  
  - 提供更方便的服務，讓用戶輕鬆使用訓練好的模型！  

### 3️⃣ Docker 化你的應用  
- **工具**：Docker  
  - 使用 Docker 技術將 API 製作成 image 和 container。  
  - 確保應用可以在任何伺服器上部署，無需擔心環境配置和依賴問題！
